# PayPal – הענקת יהלומים לתורמים

## מצב נוכחי

- "תמוך במפתח" בתפריט → קישור תשלום PayPal.
- **הענקת יהלומים ידנית**: אדמין נכנס ל־/admin, בוחר משתמש, לוחץ "הענק 2000 (תורם)".

## הענקה אוטומטית (עתידי)

דרוש: **שרת backend** שמקבל Webhook מ־PayPal אחרי תשלום, מאמת חתימה, ומעדכן ב־Supabase את `profiles.diamonds` לפי זיהוי התורם (אימייל או קוד).  
ללא שרת אי אפשר לוודא שתשלום בוצע; הענקה אוטומטית חייבת לעבור דרך backend.
